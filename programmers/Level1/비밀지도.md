### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(n, arr1, arr2) {
  let answer = [];
  let newArr1 = [];
  let newArr2 = [];

  for (let item of arr1) {
    newArr1.push(item.toString(2).padStart(n, 0));
  }

  for (let item2 of arr2) {
    newArr2.push(item2.toString(2).padStart(n, 0));
  }

  for (let i = 0; i < n; i++) {
    let newAns = '';
    for (let j = 0; j < newArr1[i].length; j++) {
      newAns += newArr1[i][j] * 1 + newArr2[i][j] * 1 != 0 ? '#' : ' ';
    }
    answer.push(newAns);
  }

  return answer;
}
```

이진수로 바꾼 후 합을 더해서 새로운 배열을 만들었다.  
padStart를 통해 빈자리에 0을 넣어줬다.

### str.padStart(targetLength [, padString])

- targetLength : 목표 문자열 길이
- padString Optional : 현재 문자열에 채워넣을 다른 문자열.
