### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(numbers) {
  var answer = [];
  const len = numbers.length;

  for (let i = 0; i < len; i++) {
    for (let j = i + 1; j < len; j++) {
      const sum = numbers[i] + numbers[j];
      answer.indexOf(sum) == -1 ? answer.push(sum) : answer;
    }
  }

  return answer.sort((x, y) => x - y);
}
```

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/68644/solution_groups?language=javascript

```javascript
function solution(numbers) {
  const temp = [];

  for (let i = 0; i < numbers.length; i++) {
    for (let j = i + 1; j < numbers.length; j++) {
      temp.push(numbers[i] + numbers[j]);
    }
  }

  const answer = [...new Set(temp)];

  return answer.sort((a, b) => a - b);
}
```

중복값을 없애기 위해 Set객체를 사용했다.

### Set 객체

자료형에 관계 없이 원시 값과 객체 참조 모두 유일한 값을 저장가능
