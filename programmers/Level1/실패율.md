### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(N, stages) {
  let answer = [];

  let totalCnt = stages.filter((e) => e == N + 1).length;

  for (let i = N; i > 0; i--) {
    let count = stages.filter((e) => e == i).length;
    totalCnt += count;
    answer.push(count == 0 && totalCnt == 0 ? 0 : count / totalCnt);
  }
  answer.reverse();
  let result = checkIndex(N, answer);
  return result;
}

function checkIndex(N, answer) {
  let result = [];
  let num = 0;

  while (N > num) {
    let max = answer.indexOf(Math.max.apply(null, answer));

    result.push(max) + 1);
    answer[max] = -1;
    num++;
  }
  return result;
}
```

`let totalCnt..` 스테이지를 다 깬경우를 초기값으로 잡는다.  
첫번째 for문으로 높은 스테이지 순서대로 개수를 구한 후 totalCnt로 나눠준다. (구한 개수는 totalCnt에 더해줌)  
`answer.reverse();` 높은 스테이지 순서대로였으니 반대로 정렬해준다.  
`checkIndex`함수는 실패율이높은 index를 찾아서 정렬하여 새로운 배열에 넣어주는함수이다.

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/42889/solution_groups?language=javascript

```javascript
function solution(N, stages) {
  let stageCount = new Array(N);
  let failPer = [];
  stageCount.fill(0);

  let usersNum = stages.length;
  for (let val of stages) {
    stageCount[val - 1]++;
  }

  for (let i = 0; i < N; i++) {
    if (usersNum === 0) {
      failPer.push({ failper: 0, stageNum: i + 1 });
    } else {
      failPer.push({ failper: stageCount[i] / usersNum, stageNum: i + 1 });
    }
    usersNum -= stageCount[i];
  }

  failPer.sort((a, b) => {
    if (a.failper === b.failper) {
      return a.stageNum - b.stageNum;
    }
    return b.failper - a.failper;
  });

  return failPer.map((item) => {
    return item.stageNum;
  });
}
```

object를 정렬하는 법을 모르겠어서 `checkIndex`함수를 사용하였는데 `failPer.sort((a, b) => {` 이 부분을 보면 object정렬하는 법을
알 수 있다.
