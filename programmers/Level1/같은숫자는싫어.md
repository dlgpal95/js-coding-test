### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(arr) {
  return arr.filter((e, i) => {
    return e != arr[i - 1];
  });
}
```

function solution(arr) {
  return arr.filter((val, index) => val != arr[index + 1]);
}
