### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(d, budget) {
  var answer = 0;

  d.sort((x, y) => x - y);

  let sum = 0;

  for (let i = 0; i < d.length; i++) {
    sum += d[i];
    if (sum == budget) {
      answer = i + 1;
      break;
    } else if (sum > budget) {
      answer = i;
      break;
    }
  }

  if (budget > sum) answer = d.length;

  return answer;
}
```

`if (budget > sum) answer = d.length;` 만약 예산이 훨씬 큰 경우를 생각을 못해서..고생했다..

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12982/solution_groups?language=javascript

```javascript
function solution(d, budget) {
  let answer = 0;
  let money = 0;
  d.sort((a, b) => a - b).forEach(function (val) {
    money += val;
    if (money <= budget) {
      answer++;
    }
  });
  return answer;
}
```

return 할때 해당 인덱스를 리턴하지않고 `answer++`를 해줬으면 if문을 2개나 줄일 수 있었다..

### :heavy_check_mark: 참고하면 좋은 다른 답안2

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12982/solution_groups?language=javascript

```javascript
function solution(d, budget) {
  var answer = 0;
  d = d.sort(function (a, b) {
    return a - b;
  });
  for (var i = 0; i < d.length, 0 < budget; i++) {
    budget = budget - d[i];
    if (budget >= 0) {
      answer++;
    }
  }
  return answer;
}
```

예산에서 빼는 방식으로 하면 변수 하나를 안써도된다.
