## 크레인 인형뽑기 게임

### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(board, moves) {
  var answer = 0;
  let stack = [];

  for (let val of moves) {
    for (let i = 0; i < board[0].length; i++) {
      if (board[i][val - 1] != 0) {
        if (board[i][val - 1] == stack[stack.length - 1]) {
          stack.pop();
          answer += 2;
        } else {
          stack.push(board[i][val - 1]);
        }
        board[i][val - 1] = 0;
        break;
      }
    }
  }
  return answer;
}
```

이중 for문을 이용해서 빼야하는 위치를 배열의 개수대로 for문을 돌려줬다.

stack을 이용하여서 배열에서 뺀 값이랑 stack맨위에 있는 값을 비교하여(0이 아닐때)  
같으면 stack맨위에 값을 빼주고 다르면 stack에 넣어줬다.

`board[i][val - 1] = 0;`이 부분을 else부분에만 적어서 고생했다..
