### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(s) {
  let answer = "";
  const num = parseInt(s.length / 2);
  answer =
    s.length % 2 == 0
      ? [...s].slice(num - 1, num + 1).join("")
      : [...s].slice(num, num + 1).join("");
  return answer;
}
```

짝/홀수를 구별한 후 slice함수로 잘라줬다.

### slice(start[, end])

- start: 추출 시작점에 대한 인덱스
- end: 추출을 종료할 기준 인덱스 (end를 제외하고 그 전까지의 요소만 추출)
- 반환값: 추출한 요소를 포함한 새로운 배열

  return s.length % 2 === 1 ? s[mid] : s[mid - 1] + s[mid];

```javascript
function solution(s) {
  return s.length % 2 == 0
    ? s.substr(s.length / 2 - 1, 2)
    : s.substr(Math.floor(s.length / 2), 1);
}
```
