## 나누어 떨어지는 숫자 배열

<hr/>

### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(arr, divisor) {
  let arry = arr.filter((e) => e % divisor === 0);
  return arry.length != 0 ? arry.sort((x, y) => x - y) : [-1];
}
```

<hr/>

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12910/solution_groups?language=javascript

```javascript
function solution(arr, divisor) {
  var answer = [];
  arr.map((o) => {
    o % divisor === 0 && answer.push(o);
  });
  return answer.length ? answer.sort((a, b) => a - b) : [-1];
}
```

배열 내장함수 map을 사용한 코드이다. 개인적으로 filter가 더 편하다.
