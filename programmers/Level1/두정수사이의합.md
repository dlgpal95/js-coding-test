## 두 정수 사이의 합

<hr/>

### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(a, b) {
  let answer = 0;
  if (a === b) {
    return a;
  }
  const maxNum = Math.max(a, b);
  const minNum = Math.min(a, b);
  for (let i = minNum; i <= maxNum; i++) {
    answer += i;
  }
  return answer;
}
```

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12912/solution_groups?language=javascript

```javascript
function adder(a, b, s = 0) {
  for (var i = Math.min(a, b); i <= Math.max(a, b); i++) s += i;
  return s;
}
```

변수선언 안하고 바로 넣은 코드 훨씬 깔끔하다.  
같을 숫자일때 조건을 안해도 되는 거 였다.

function adder(a, b) {
  var result = 0;
  return ((a + b) * (Math.abs(b - a) + 1)) / 2;
