### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(n) {
  let answer = 0;
  let stack = [];
  let newNum = n;

  while (true) {
    if (newNum < 3) {
      stack.push(newNum);
      break;
    }
    let num = newNum % 3;
    stack.push(num);
    newNum = parseInt(newNum / 3);
  }
  const len = stack.length;

  for (let i = 0; i < len; i++) {
    answer += stack.pop() * Math.pow(3, i);
  }
  return answer;
}
```

stack을 이용해서 while문으로 3진법을 바꿔주고 reverse를 안시키고 `pop()`을 하여 10진법을 사용해줬다.  
`parseInt(newNum / 3)`을 이용해서 몫을 구해줬다.  
`pop()`하면 stack의 길이가 바뀌기때문에 `const len = stack.length;`으로 미리 저장해줬다.

#### Math.pow(base, exponent) : 거듭제곱

- base : 기본 번호
- exponent : 올리는 데 사용되는 지수

```javascript
const solution = (n) => {
  return parseInt([...n.toString(3)].reverse().join(""), 3);
};
```

`[...n.toString(3)]` ...n을 이용해서 배열로 바꿔준다.  
` toString(3)` 3진수로 변경한다. (toString이 이런기능 있는 것을 처음 알았다..) `reverse().join("")`배열의 순서를 바꿔주고 join함수를 이용하여 다시 숫자로 바꿔준다. `parseInt(~, 3);``` 3진수 -> 10진수로 바꿔준다.

```javascript
function solution(n) {
  const answer = [];
  while (n !== 0) {
    answer.unshift(n % 3);
    n = Math.floor(n / 3);
  }
  return answer.reduce((acc, v, i) => acc + v * Math.pow(3, i), 0);
}
```

`unshift`을 이용해서 reverse()를 쓰지않아도 되는 코드이다.  
`Math.floor`을 이용해서 몫을 구해줬다.  
`reduce`를 이용해서 내코드의 마지막 for문을 깔끔하게 정리해줬다..
