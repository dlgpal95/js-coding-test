## 소수만들기

<hr/>

### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(nums) {
  let answer = 0;
  const len = nums.length;

  for (let i = 0; i < len; i++) {
    for (let j = i + 1; j < len; j++) {
      for (let k = j + 1; k < len; k++) {
        answer += checkNum(nums[i] + nums[j] + nums[k]);
      }
    }
  }
  return answer;
}
// 소수인지 체크하는 함수 : 소수면 1 소수가 아니면 0 return
function checkNum(n) {
  console.log(n);
  let check = false;
  if (n % 2 != 0) {
    for (let j = 3; j <= Math.sqrt(n); j += 2) {
      if (n % j == 0 && j != n) {
        check = true;
        break;
      }
    }
  } else {
    check = true;
  }
  return check ? 0 : 1;
}
```

`checkNum()`는 소수인지 체크하는 함수이다. 소수면 1 소수가 아니면 0 return한다.
