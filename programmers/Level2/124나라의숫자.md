## 124 나라의 숫자

<hr/>

### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(n) {
  let answer = '';

  while (n > 0) {
    switch (n % 3) {
      case 1:
        answer = '1' + answer;
        n = Math.floor(n / 3);
        break;
      case 2:
        answer = '2' + answer;
        n = Math.floor(n / 3);
        break;
      case 0:
        answer = '4' + answer;
        n = n / 3 - 1;
        break;
    }
  }

  return answer;
}
```

숫자를 3으로 나눴을때 아래와 같이 나타난다.

1. 나머지 1 : '1'
2. 나머지 2 : '2'
3. 나머지 0 : '4'

나머지가 0일 때(3의 배수일 때), 앞자리가 바뀌거나 숫자의 길이가 바뀌기 때문에 -1을 해준다.
