### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(arr1, arr2) {
  var answer = [];

  arr1.map((e, i) => {
    let arrSum = [];

    for (let k = 0; k < arr2[0].length; k++) {
      let sum = 0;
      e.map((el, j) => {
        sum += el * arr2[j][k];
      });
      arrSum.push(sum);
    }
    answer.push(arrSum);
  });

  return answer;
}
```

맨처음 arr1을 for문을 해주고  
각 행을 또 for문을 arr2의 행의 개수만큼 돌려주면서 계산해준다.

즉 arr1행들의 계산을 해주고 arrSum에 담고  
총 값을 answer에 담아서 return 해준다.

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12949/solution_groups?language=javascript

```javascript
function solution(arr1, arr2) {
  return arr1.map((row) => arr2[0].map((x, y) => row.reduce((a, b, c) => a + b * arr2[c][y], 0)));
}
```

내 코드에서 sum에 넣어주는 값들을 reduce로 해결할수 있었다.
