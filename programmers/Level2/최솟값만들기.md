## 최솟값 만들기

<hr/>

### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(A, B) {
  let answer = 0;

  A.sort((a, b) => a - b);
  B.sort((a, b) => b - a);

  A.map((e, i) => {
    answer += e * B[i];
  });

  return answer;
}
```

최소값은 A의 가장큰 값과 B의 가장작은값을 곱하거나 B의 가장큰 값과 A의 가장작은값을 곱하는 것이다.  
그에 따라 서로 정렬을 해주고 곱했다.

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12941/solution_groups?language=javascript

```javascript
function solution(A, B) {
  A.sort((a, b) => a - b);
  B.sort((a, b) => b - a);
  return A.reduce((total, val, idx) => total + val * B[idx], 0);
}
```

곱셈을 할때 reduce를 생각해보자!
