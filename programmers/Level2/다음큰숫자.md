### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(n) {
  let answer = 0;
  let answerNum = n + 1;
  let oneCount = [...n.toString(2)].filter((e) => e == 1).length;

  while (true) {
    if (oneCount == [...answerNum.toString(2)].filter((e) => e == 1).length) {
      return answerNum;
    }
    answerNum++;
  }

  return answer;
}
```

toStirng으로 2진법으로 바꿔주고 filter를 이용해서 1의 개수를 구해줬다.  
filter때문에 속도가 좀 느린거 같다.

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12911/solution_groups?language=javascript

```javascript
function solution(n, a = n + 1) {
  return n.toString(2).match(/1/g).length == a.toString(2).match(/1/g).length ? a : solution(n, a + 1);
}
```

정규식과 match 함수를 이용했는데 확실히 속도가 빠르다.  
재귀함수를 써서 코드도 짧아졌다.
