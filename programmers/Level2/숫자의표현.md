### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(n) {
  let answer = 0;

  for (let i = 1; i <= n; i++) {
    answer += check(i, n);
  }
  return answer;
}

function check(i, n) {
  let sum = i;
  while (sum <= n) {
    if (sum == n) {
      return 1;
    }
    sum += ++i;
  }
  return 0;
}
```

1부터 n 까지 for문을 돌려서 check함수에서 나온 값을 더해준다.
check함수는 받아온 i를 ++해서 계속 sum에 더하는 함수이다.

1. sum 이 n이랑 같으면 `return 1`을 해준다.
2. sum 이 n보다 크면 while문을 벗어나 `return 0`을 해준다.

```javascript
function expressions(num) {
  var answer = 0;

  for (var i = 1; i <= num; i++) {
    if (num % i == 0 && i % 2 == 1) answer++;
  }
  return answer;
}
```
